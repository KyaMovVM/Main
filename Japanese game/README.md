# Japanese Game / Visual Novel

Проект японской визуальной новеллы (VN) на JavaScript.

## Структура

```
Japanese game/
├── index.html       # Главная страница
├── index.js         # Основной скрипт
├── novel.js         # Логика визуальной новеллы
├── dialogues.md     # Диалоги
├── doc/             # Документация
└── styles/          # Стили (опционально)
```

## Развертывание

```bash
# Локальный сервер
python -m http.server 8000

# Или с Node.js
npx http-server
```

Затем откройте http://localhost:8000

## Технологии

- HTML5
- JavaScript (ES6+)
- CSS3
- Markdown для диалогов
